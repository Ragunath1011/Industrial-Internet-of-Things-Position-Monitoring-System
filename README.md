# Industrial IoT Position Monitoring System

## Overview
This project monitors the position and orientation of an object using an ESP32, MPU6050 sensor, OLED display, Wokwi simulation, and ThingSpeak cloud platform.

## Hardware
- ESP32
- MPU6050
- SSD1306 OLED

## Software
- Arduino IDE
- Wokwi
- GitHub

## Cloud
- ThingSpeak

## Features
- Real-time accelerometer monitoring
- Real-time gyroscope monitoring
- OLED display
- Cloud data upload using ThingSpeak
- Wokwi simulation
